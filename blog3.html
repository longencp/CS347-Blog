This week we continuted our learning about CSS and designing web pages. One of the parts we focused most on was flexbox. I learned that flexbox is a useful tool for designing webpages because it makes it much easier to control the layout of different elements on a page. While we were learning about flexbox I got more interested in fun design elements and I came across animations in CSS.

In my reasearch I came across an article from w3schools which explains how these work. Using the @keyframes element, developers can choose the start and end state of an animation. In this example, I am changing the color from purple to yellow and increasing the height from 100px to 500px. Developers can choose which animation they want to use using animation-name and the name of the animation they developed using the @keyframes element. Developers can also change the duration of the animation using animation-duration. After the animation is completed, the animated object will return to its original state. The following code demonstrates how this webpagewas able to animate the rectangle below.

    div {
        width: 50px;
        height: 100px;
        background-color: purple;
        animation-name: animation;
        animation-duration: 5s;
    }
    @keyframes animation {
        from {height: 100px; background-color: purple;}
        to {height: 500px; background-color: yellow;}
