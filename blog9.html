Blog Post #9: 11/19/21

This week we continued our study of react and JSX. It has been really interesting learning more about how to use react and how it is different from the topics we covered at the beginning of the semester. I became more curious about how react changes things that are also in HTML and began to research it.

In my research, I came across an article which explained fragments vs divs. Divs are not always useful and can require unnecessary syntax. As a solution, react has something called fragments. Fragments can replace wrapper divs and can hold a list of html elements. For example this code from the article using divs: 

        <table>
            <tr>
              <div>
                <td>Hello</td>
                <td>World</td>
              </div>
            </tr>
          </table>
could be re-written using this 
            <React.Fragment>
                <td>Hello</td>
                <td>World</td>
              </React.Fragment>
