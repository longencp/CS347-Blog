Blog Post #1

This week I started learning about HTML and how to use it. I learned many different element tags used to format the webpage such as divs, headers, paragraphs, and more. As I was doing the reading for this week, I came across the section about screen readers and I became very interested in how specifically to optomize a webpage for a screen reader.

I did some research and ended up finding an article from WebAIM about making webpages more accessible for screen readers which ended up being very helpful. One helpful thing that I learned was that using heading and paragraph tags are important ways to organize main ideas of the page, as users can jump between these. Another aspect of html which is particularly helpful for people using screen readers in the lang attribute for the span tag. This can be used to specify the language that is being read, so that if the webpage inclues sections in other languages, the reader knows how to pronounce them. For example, if I wanted to say If I wanted to say hello in Spanish, I would say hola and I wanted to make sure that the hola was pronounced correctly, then I could specify it like in the code below: 

        <p>If I wanted to say hello in Spanish, I would say <span lang="es">hola</span></p>
